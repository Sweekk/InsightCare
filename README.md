  Project Overview

 The AI-Powered Doctor Dashboard is a secure, multi-user HealthTech platform designed to help doctors monitor patient health, assess risk levels, and receive AI-generated clinical insights.

 The system enables doctors to manage their own patients, track high-risk cases, and make informed medical decisions using AI-driven analysis â€” all within a clean, professional medical interface.
